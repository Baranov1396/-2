Основы создания интерактивных текстов

В этом разделе мы рассмотрим основные принципы создания интерактивного текста и пример их реализации.

(База данных) Результаты Викторины:

Обучающие модули

Сделать текст красным

Этот текст станет красным при нажатии на кнопку.

element.style.color = 'red';

Сделать текст жирным

Этот текст станет жирным при нажатии на кнопку.

element.style.fontWeight = 'bold';

Поменять текст

Этот текст изменится при нажатии на кнопку.

element.innerText = 'Новый текст';

Сделать текст мигающим

Этот текст будет мигать при нажатии на кнопку.

@keyframes blink { 50% { opacity: 0; } } element.style.animation = 'blink 1s steps(1, start) infinite';

Телепортировать текст

Этот текст телепортируется при нажатии на кнопку.

element.style.transform = 'translateX(100px)';
